## Student record manager system

This is a system made based on java Object-Orinted Programming.This application manages student record using Java File I/O and streams.

The system allows user to add ,search,update ,Delete and Display all students.

It also genertes reports and stores data using text files, binary files, and object serialization.
## Features
**add student**

**search student by ID**

**update student information**

**Delete student**

**Display all students**

**Genrate report**

**Save Data to Text File**

**Save Data to Binary File**

**Save Data using Object Serialization**

**Display File Information**

**Create Backup using Buffered Streams**

**Exception Handling**
